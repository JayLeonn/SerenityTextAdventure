# SerenityTextAdventure
A text adventure (look up zork if you dont know what a text adventure is) game made as a School Project during my 1st university semester. Written with Python

HOW TO INSTALL:
1. Download python 3.3.5 or newer (only tested with python 3.3.5) at https://www.python.org/download/releases/3.3.5/
2. Download mySQL connector for python at http://dev.mysql.com/downloads/connector/python/.
3. Download mariadb at https://downloads.mariadb.org/
4. Open MySQL Client and log in. Then type "\. 'location'\'of'\'the'\serenitysql.sql"
5. Double click serenitypy.py to play the game
